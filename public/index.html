Why Choose Konoha Land API?

Built for developers, by developers. Everything you need to integrate Indonesian regional data.

Lightning Fast

Optimized queries and global CDN ensure sub-100ms response times worldwide.

No Authentication

Start building immediately. No API keys, no rate limits, no signup required.

Complete Dataset

2025 data: 38 provinces, 514 regencies, 7,285 districts, 83,762 villages with 100% postal code coverage.

Advanced Features

Search, sort, filter, and paginate. Includes sortBy & sortDirection parameters for custom ordering.

Backward Compatible

Drop-in replacement for legacy APIs. Your existing code will work.

Open Source

Fully open source. Fork it, modify it, deploy your own instance.

Try It Live

Test API endpoints directly in your browser. No setup required.
